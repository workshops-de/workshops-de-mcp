Workshops.DE MCP Server auf Vercel

Dies ist ein Model Context Protocol (MCP) Server für die Workshops.DE API, gehostet auf Vercel mit nativer MCP-Unterstützung.

MCP Endpoint

https://workshops-mcp.vercel.app/api/mcp

Dieser Endpoint unterstützt das Vercel Streamable HTTP Transport Format für MCP.

Verfügbare Tools

  • list_courses - Zeigt alle verfügbaren Kurse
  • get_course_events - Zeigt Events für einen bestimmten Kurs
  • list_trainers - Zeigt alle Trainer
  • get_course_trainers - Zeigt Trainer für einen bestimmten Kurs
  • list_events - Zeigt alle kommenden Schulungen
  • get_event - Zeigt Details zu einem bestimmten Event

MCP Testen

Nutze den MCP Inspector, um den Server zu testen: npx @modelcontextprotocol/inspector@latest https://workshops-mcp.vercel.app

MCP Client Konfiguration

// Für Cursor (.cursor/mcp.json)
{
  "mcpServers": {
    "workshops-de": {
      "url": "https://workshops-mcp.vercel.app/api/mcp"
    }
  }
}

// Für Claude Desktop
{
  "mcpServers": {
    "workshops-de": {
      "url": "https://workshops-mcp.vercel.app/api/mcp",
      "transport": "http"
    }
  }
}

// 1. MCP Inspector verwenden (empfohlen für Tests)
